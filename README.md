**Task 1 | Quantium Data Analytics job simulation**

## Overview

This project examines the purchashing and behaviour trends of customers to understand their segments. By defining metrics of interest to drawn insights on who spends on chips and what drives spends for each customer segment. With objective to deliver strategy dased on the findings for commercial application.

## Project Structure

```text
chips-customer-analytics/
├── README.md		    # This file
├── Data/
│   └── bronze/         # transcation and purchase behaviour data
├── notebooks/          # experimentation notebooks
└── requirements.txt    # required libraries
```

## Quickstart

Prereqs: **Python 3.10+**

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Open and run:
- `notebooks/pre-processing.ipynb`
